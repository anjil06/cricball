# 🏏 CricBall

A Flutter-based Cricket Scoring Application that enables users to score cricket matches digitally. The application provides an intuitive interface for recording runs, wickets, overs, player statistics, and match summaries.

## 📱 Features

* Live cricket score management
* Ball-by-ball scoring
* Over and wicket tracking
* Player statistics
* Match summary
* Responsive Flutter UI
* Local data persistence
* Simple and user-friendly interface

## 🛠️ Tech Stack

* Flutter
* Dart
* Material Design
* SQLite / Shared Preferences (if applicable)
* Git & GitHub

## 🚀 Getting Started

### Prerequisites

* Flutter SDK
* Android Studio or VS Code
* Android Emulator or Physical Device

### Installation

```bash
git clone https://github.com/anjil06/cricball.git

cd cricball

flutter pub get

flutter run
```
